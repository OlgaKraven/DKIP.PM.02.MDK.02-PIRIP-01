    Описание дисцплины ПИРИП

    Цели изучения дисциплины

    Содержание дисциплины

    • Дисциплина: Проектирование и разработка интерфейсов пользователя
    • Длительность: 1 семестр
    • Количество лабораторных: 7 лабораторных работ
    • Баллы за лабораторные: 30 баллов
    • ИМ: БЕЗ ИМ

    Вариант на семестр

    Описание: У каждого студента свой вариант предметной области, прикрепленный в ЛМС. Предметная область остается на весь семестр начиная с лабораторной работы №2.

    Цель изучения дисциплины

    Овладение концепцией и принципами разработки интерфейса пользователя.

    Задачи изучения дисциплины

    • формирование знаний о нормах и правилах выбора стилистических решений при разработке веб-интерфейсов, современных методиках проектирования графического пользовательского интерфейса (GUI), принципах эргономики, технической эстетики и визуальной коммуникации в цифровой среде;

    • формирование знаний о требованиях и нормах подготовки, обработки и оптимизации изображений для использования в информационно-телекоммуникационной сети «Интернет», включая форматы графики, параметры качества, адаптивность и производительность;

    • формирование умений разрабатывать дизайн веб-приложений в соответствии с техническим заданием, стандартами отрасли и требованиями заказчика, обеспечивая целостность визуальной концепции и соответствие целевому рынку;

    • формирование умений выбирать и обосновывать дизайнерские решения с учётом особенностей целевой аудитории, задач продукта, принципов UX/UI-дизайна и современных тенденций веб-разработки;

    • формирование умений создавать дизайн веб-приложений с использованием промежуточных эскизов (wireframes), прототипов и макетов, учитывая требования к эргономике, доступности (accessibility) и пользовательскому опыту;

    • формирование практического опыта разработки интерфейса пользователя веб-приложений с применением современных стандартов, технологий и инструментов проектирования;

    • формирование практического опыта создания, использования и оптимизации графических элементов и изображений для веб-приложений с учётом требований производительности и визуального качества;

    • формирование уровня знаний, умений и практического опыта деятельности в рамках подготовки кадров для цифровой экономики, ориентированной на развитие современных веб-технологий и цифровых сервисов;

    • формирование профессиональных компетенций, соответствующих государственным стандартам и требованиям к разработке дизайна веб-приложений, а также лучшим практикам в области информационных технологий и цифрового проектирования.

    Темы курса

    Тема 1. Основы HTML5 и структурная разметка веб-интерфейса (2 ч)

    Содержание:

    • Введение в веб-разработку как проектирование интерфейса
    • Язык разметки HTML5: синтаксис и структура документа
    • Семантические элементы HTML5
    • Гиперссылки и навигация
    • Использование изображений и медиа
    • Форматирование текста и фоновых элементов
    • Элементы и атрибуты HTML5
    • Основы доступности (alt, label, lang)

    Связь с лабораторными: → ЛР №1 (базовая структура страницы) → подготовка к ЛР №2 (форма как элемент интерфейса)

    Тема 2. Структурные элементы интерфейса: списки, таблицы, формы (1 ч)

    Содержание:

    • Списки как элементы навигации и структуры контента
    • Таблицы как способ представления данных интерфейса
    • Формы как инструмент взаимодействия пользователя
    • Валидация HTML5
    • Встраивание контента (iframe)
    • Архитектура страницы (header/nav/main/section/footer)

    Связь с лабораторными: → ЛР №2 (UX-ориентированная форма) → основа для интерактивности (ЛР №5)

    Тема 3. CSS как инструмент визуального проектирования интерфейса (1 ч)

    Содержание:

    • Подключение CSS
    • Каскадность и наследование
    • Модель коробки (box model)
    • Цветовые модели и фоновые свойства
    • Базовые стили текста
    • Псевдоклассы и псевдоэлементы

    Связь с лабораторными: → ЛР №3 (создание дизайн-системы)

    Тема 4. Веб-стандарты, семантика и архитектура страницы (1 ч)

    Содержание:

    • Роль стандартов W3C
    • HTML5 vs устаревшие подходы
    • Разделение структуры, представления и поведения
    • SEO-ориентированная разметка
    • Основы accessibility (WCAG)
    • Поддержка браузерами

    Связь с лабораторными: → повышение качества ЛР №1–3 → подготовка к профессиональной вёрстке (ЛР №4)

    Тема 5. Селекторы и продвинутые возможности HTML5 и CSS (1 ч)

    Содержание:

    • Типы CSS-селекторов (атрибутные, вложенные, комбинированные)
    • CSS-переменные (custom properties)
    • Позиционирование элементов
    • Transition и базовые анимации
    • Responsive-подход (media queries)
    • Интерактивные HTML5-элементы

    Связь с лабораторными: → ЛР №3 (UI-компоненты) → ЛР №7 (hero-блок и анимации)

    Тема 6. CSS3 и адаптивная вёрстка интерфейса (1 ч)

    Содержание:

    • Flexbox
    • CSS Grid
    • Mobile-first подход
    • Breakpoints
    • Sticky/Fixed элементы
    • Clamp(), min(), max()
    • Адаптивная типографика
    • Оптимизация отображения

    Связь с лабораторными: → ЛР №4 (адаптивная вёрстка) → ЛР №6 (оптимизация интерфейса)

    Тема 7. Индустриальные инструменты: CSS-фреймворки и препроцессоры (1 ч)

    Содержание:

    • CSS-фреймворки (Bootstrap, Tailwind — обзор)
    • Компонентный подход
    • Препроцессоры (LESS / SCSS) — переменные, вложенность
    • Повторное использование стилей
    • Шаблоны CMS и типовые решения
    • Быстрое прототипирование интерфейса

    Связь с лабораторными: → усиление ЛР №3–4 → подготовка к ЛР №7 (UI-баннер)

    Описание лабораторных работ

    № Название лабораторной работы Описание лабораторной работы Баллы
    1 Применение тегов HTML при создании web-страниц Формирование навыков структурной разметки HTML-документа. Создание семантически корректной страницы с использованием заголовков, абзацев, списков, ссылок, таблиц и изображений. Организация структуры проекта, соблюдение стандартов HTML5 и базовых принципов доступности. 4
    2 Проектирование и реализация пользовательской формы с UX-валидацией Разработка пользовательской формы с продуманной логикой взаимодействия. Реализация клиентской валидации, обработка ошибок, визуальная индикация состояний полей (focus/error/success), использование атрибутов accessibility. Организация UX-поведения при отправке данных без перезагрузки страницы. 4
    3 Разработка дизайн-системы интерфейса с использованием CSS Создание визуальной системы интерфейса: разработка цветовой палитры, типографической шкалы, системы отступов и повторно используемых UI-компонентов (кнопки, карточки, поля ввода). Использование CSS-переменных, псевдоклассов, адаптивных единиц измерения и реализация единого визуального стиля. 4
    4 Адаптивная и семантически корректная вёрстка интерфейса Проектирование адаптивного пользовательского интерфейса с использованием Flexbox и CSS Grid. Реализация mobile-first подхода, настройка breakpoints, построение навигации и структурных блоков страницы. Обеспечение корректного отображения интерфейса на различных устройствах. 4
    5 Интерактивность интерфейса с использованием JavaScript Реализация динамического поведения интерфейса: обработка событий, изменение DOM-структуры, управление классами, создание модальных окон, вкладок и фильтров. Использование localStorage для сохранения состояния интерфейса и организация взаимодействия пользователя с элементами страницы. 4
    6 Оптимизация графики и визуальной производительности интерфейса Повышение производительности пользовательского интерфейса за счёт оптимизации изображений и графики. Использование форматов WebP, SVG, lazy loading, адаптивных изображений (srcset), минимизация ресурсов. Анализ показателей производительности и улучшение скорости загрузки страницы. 4
    7 Разработка интерактивного UI-баннера (Hero Section) Проектирование и реализация hero-блока как ключевого элемента интерфейса. Создание визуальной композиции, типографической иерархии, CTA-кнопок и анимаций. Обеспечение контрастности, доступности и адаптивности. Интеграция интерактивных эффектов для повышения вовлечённости пользователя. 6

    Полезные ресурсы для изучения веб-разработки

    Официальная документация и стандарты

    • MDN Web Docs — JavaScript (RU)
      https://developer.mozilla.org/ru/docs/Web/JavaScript
      Официальная документация по JavaScript с примерами, описанием синтаксиса, встроенных объектов и Web API.

    • ECMAScript (ECMA-262)
      https://tc39.es/ecma262/ Спецификация языка JavaScript, описывающая формальный стандарт ECMAScript.

    Учебные материалы и справочники

    • JavaScript.info (Илья Кантор)
      https://javascript.info/ Подробный русскоязычный учебник по JavaScript: основы языка, работа с DOM, асинхронное программирование, объектная модель.

    • Eloquent JavaScript
      https://eloquentjavascript.net/ Онлайн-книга, ориентированная на понимание принципов работы языка и развитие алгоритмического мышления.

    • You Don’t Know JS (Yet)
      https://github.com/getify/You-Dont-Know-JS Серия книг, посвящённых внутренним механизмам JavaScript и продвинутым аспектам языка.

    Практика и интерактивные среды

    • Codewars (JavaScript)
      https://www.codewars.com/ Платформа для решения алгоритмических задач на JavaScript с автоматической проверкой решений.

    • JSFiddle
      https://jsfiddle.net/ Онлайн-песочница для экспериментов с JavaScript, HTML и CSS.

    • CodePen
      https://codepen.io/ Среда для создания и демонстрации фронтенд-примеров и небольших web-компонентов.

    • StackBlitz
      https://stackblitz.com/ Онлайн-IDE для разработки JavaScript- и Node.js-проектов в браузере.

    Статьи, обзоры и профессиональное сообщество

    • Habr — раздел JavaScript
      https://habr.com/ru/hub/javascript/ Публикации, разборы практических кейсов, обзоры инструментов и обсуждение современных подходов к разработке.

    • Hexlet Blog
      https://ru.hexlet.io/blog Аналитические статьи по JavaScript, архитектуре приложений и качеству кода.

    Дополнительные ресурсы для изучения JavaScript

    • learn.javascript.ru
      https://learn.javascript.ru/ Русскоязычный учебный ресурс, подробно раскрывающий основы и продвинутые возможности JavaScript: синтаксис, работа с DOM, события, асинхронность, объектная модель.

    • W3Schools — JavaScript
      https://www.w3schools.com/js/ Справочник и практические примеры по JavaScript, ориентированные на быстрое освоение базовых конструкций языка и работы с браузером.

    • Code.mu — JavaScript
      https://code.mu/ru/javascript/book/prime/ Русскоязычный учебник по JavaScript с последовательной подачей материала и практическими заданиями для закрепления навыков.
